## 建议测试顺序

1. 用需要 3D Secure 的 Stripe 测试卡验证认证成功/失败两条路径。
2. 用 Stripe Test Clock 验证周期结束、取消后到期和续期。
3. 验证 Stripe webhook 事件与回跳同步的边界：事件正常、事件延迟、重复事件。
4. 测试 Checkout 会话过期、Stripe API 临时失败和多个标签页同时发起支付。
